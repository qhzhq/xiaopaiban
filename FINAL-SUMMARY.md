## 开发指南

### 环境准备
1. 安装Node.js >= 18
2. 安装npm >= 9
3. 安装Git

### 快速开始
```bash
# 克隆项目
git clone https://github.com/qhzhq/xiaopaiban.git

# 进入项目目录
cd xiaopaiban

# 安装依赖
npm install

# 启动H5端开发
npm run dev:h5

# 或者使用启动脚本
./start.sh
```

### 开发命令
```bash
# H5端开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin

# 支付宝小程序开发
npm run dev:mp-alipay

# 构建H5端
npm run build:h5

# 构建微信小程序
npm run build:mp-weixin
```

## 注意事项

### 1. 平台差异
- 不同平台API支持程度不同，需要条件编译
- 部分功能在小程序端受限，需要降级处理
- 样式在不同平台可能有差异，需要测试验证

### 2. 性能考虑
- 大文本排版可能影响性能，需要优化算法
- 小程序包大小有限制，需要代码分割
- 不同设备性能差异，需要适配优化

### 3. 兼容性
- 不同平台版本兼容性不同，需要测试覆盖
- 旧版本浏览器可能不支持某些特性
- 不同设备屏幕尺寸需要响应式设计

### 4. 安全性
- 用户数据本地存储，注意隐私保护
- 文件导入导出需要安全检查
- 网络请求需要HTTPS协议
