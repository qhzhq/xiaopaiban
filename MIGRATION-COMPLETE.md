## 使用说明

### 快速开始
```bash
# 1. 克隆项目
git clone https://github.com/qhzhq/xiaopaiban.git

# 2. 进入项目目录
cd xiaopaiban

# 3. 安装依赖
npm install

# 4. 启动H5端开发
npm run dev:h5

# 或者使用启动脚本
./start.sh
```

### 支持的平台
- **H5端**: `npm run dev:h5` 或 `npm run build:h5`
- **微信小程序**: `npm run dev:mp-weixin` 或 `npm run build:mp-weixin`
- **支付宝小程序**: `npm run dev:mp-alipay` 或 `npm run build:mp-alipay`
- **其他小程序**: 支持百度、字节跳动、QQ、快手、京东、360等平台

### 开发指南
1. **代码编写**: 使用Vue 3 + TypeScript编写组件
2. **状态管理**: 使用Pinia管理应用状态
3. **样式编写**: 使用CSS变量实现主题切换
4. **API调用**: 使用uni-app x提供的API
5. **平台适配**: 使用条件编译处理平台差异

## 注意事项

### 1. 平台差异
- 不同平台API支持程度不同，需要条件编译
- 部分功能在小程序端受限，需要降级处理
- 样式在不同平台可能有差异，需要测试验证

### 2. 性能考虑
- 大文本排版可能影响性能，需要优化算法
- 小程序包大小有限制，需要代码分割
- 不同设备性能差异，需要适配优化

### 3. 兼容性
- 不同平台版本兼容性不同，需要测试覆盖
- 旧版本浏览器可能不支持某些特性
- 不同设备屏幕尺寸需要响应式设计

### 4. 安全性
- 用户数据本地存储，注意隐私保护
- 文件导入导出需要安全检查
- 网络请求需要HTTPS协议
