快速开始
============================================================

1. 克隆项目
   git clone https://github.com/qhzhq/xiaopaiban.git

2. 进入项目目录
   cd xiaopaiban

3. 安装依赖
   npm install

4. 启动开发
   npm run dev:h5

   或者使用启动脚本
   ./start.sh
