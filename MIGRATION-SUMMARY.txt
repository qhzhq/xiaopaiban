四、快速开始
============================================================

1. 克隆项目
   git clone https://github.com/qhzhq/xiaopaiban.git

2. 进入项目目录
   cd xiaopaiban

3. 安装依赖
   npm install

4. 启动H5端开发
   npm run dev:h5

   或者使用启动脚本
   ./start.sh

============================================================
五、支持的平台
============================================================

- H5端: npm run dev:h5 或 npm run build:h5
- 微信小程序: npm run dev:mp-weixin 或 npm run build:mp-weixin
- 支付宝小程序: npm run dev:mp-alipay 或 npm run build:mp-alipay
- 其他小程序: 支持百度、字节跳动、QQ、快手、京东、360等平台
- App端: 使用HBuilderX打包

七、功能特性
============================================================

核心功能:
1. 智能排版
   - 段落缩进 (0-8个字符)
   - 段间空行 (0-3行)
   - 段内换行 (1-5种模式)

2. 字符转换
   - 全角半角转换 (数字、字母、标点、空格)

3. 主题系统

   - 公众号排版
   - 纯文本清理
   - 代码注释

5. 导入导出
   - 支持TXT、Markdown、HTML格式

   - 文件导入导出

6. 统计信息
