# 小排版项目总结

## 项目概述

小排版是一款简洁好用的文本排版工具，已成功从原有的Tauri桌面应用迁移到uni-app x框架，实现了跨平台支持。

## 技术架构

### 原架构
- **前端**: 原生HTML/CSS/JavaScript
- **桌面框架**: Tauri (Rust)
- **构建工具**: 无
- **状态管理**: 无
- **平台支持**: 仅桌面端 (Windows, macOS, Linux)

### 新架构
- **前端**: Vue 3 + TypeScript
- **跨平台框架**: uni-app x
- **构建工具**: Vite
- **状态管理**: Pinia
- **平台支持**: H5 + 小程序 + App

## 功能特性

### 核心功能
1. **智能排版**
   - 段落缩进 (0-8个字符)
   - 段间空行 (0-3行)
   - 段内换行 (1-5种模式)

2. **字符转换**
   - 全角半角转换 (数字、字母、标点、空格)
   - 标点符号修正 (多种模式)
   - 波浪线修正 (全角/半角转换)

3. **主题系统**
   - 亮色主题
   - 暗色主题
   - 自动主题 (跟随系统)

4. **预设配置**
   - 小说投稿
   - 论文排版
   - 公众号排版
   - 纯文本清理
   - 代码注释

5. **导入导出**
   - 支持TXT、Markdown、HTML格式
   - 剪贴板导入导出
   - 文件导入导出

6. **统计信息**
   - 字数统计
   - 行数统计
   - 段落统计

### 辅助功能
- 快捷键支持
- 历史记录
- 设置持久化
- 自动排版
- 同步滚动

## 开发指南

### 环境准备
1. 安装Node.js >= 18
2. 安装npm >= 9
3. 安装Git

### 快速开始
```bash
# 克隆项目
git clone https://github.com/qhzhq/xiaopaiban.git

# 进入项目目录
cd xiaopaiban

# 安装依赖
npm install

# 启动H5端开发
npm run dev:h5

# 或者使用启动脚本
./start.sh
```

### 开发命令
```bash
# H5端开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin

# 支付宝小程序开发
npm run dev:mp-alipay

# 构建H5端
npm run build:h5

# 构建微信小程序
npm run build:mp-weixin
```

### 代码规范
- 使用TypeScript编写代码
- 遵循Vue 3组合式API规范
- 使用Pinia进行状态管理
- 编写清晰的注释和文档

## 部署指南

### H5端部署
1. 构建项目: `npm run build:h5`
2. 上传`dist/build/h5`目录到Web服务器
3. 配置域名和SSL证书

### 微信小程序部署
1. 构建项目: `npm run build:mp-weixin`
2. 使用微信开发者工具打开`dist/build/mp-weixin`
3. 上传代码到微信后台
4. 提交审核并发布

### App端部署
1. 使用HBuilderX打开项目
2. 选择原生App-云打包
3. 配置应用信息和证书
4. 打包并发布到应用商店

## 常见问题

### Q: 如何自定义预设？
A: 在设置面板中选择"自定义"预设，然后调整各项参数，设置会自动保存。

### Q: 如何导入文件？
A: 点击工具栏的"导入"按钮，可以选择从剪贴板导入或从文件导入（H5端支持）。

### Q: 如何切换主题？
A: 点击标题栏的主题图标，或在设置面板中选择主题模式。

### Q: 数据会保存在哪里？
A: 所有设置和历史记录都保存在本地存储中，不会上传到服务器。

### Q: 如何支持更多平台？
A: 参考uni-app官方文档，添加相应的平台配置和适配代码。
